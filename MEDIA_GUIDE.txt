          FRANKIE'S WEBSITE — IMAGE & VIDEO DIMENSIONS GUIDE
==============================================================================

Use this guide to create images/videos that fit perfectly in each section.
All media uses object-fit: cover (fills the area, crops edges if needed).
For best results, match the recommended aspect ratio.

TIP: Videos should be MP4 format. Images should be JPG/PNG/WEBP.
TIP: "cover" means media fills the container — center your subject!

==============================================================================
  VIDEO TIPS
==============================================================================

• Format:       MP4 (H.264 codec) for best browser compatibility
• Resolution:   Match the recommended sizes above
• Duration:     Short loops (5–15 seconds) work best
• Audio:        Videos auto-play MUTED — no audio needed
• Behavior:     Videos auto-play, loop, and are muted on the website
• File Size:    Keep under 5MB for fast loading (compress with HandBrake)
• Aspect Ratio: Match the section's aspect ratio to avoid cropping
